# SudokuSolver
- Compile with the make command

- launch the program with ./sudoku [options] [FILE]

    options are:
    
      -oFILE,  write result to FILE.
      
      -gSIZE,  generate a SIZE-sized grid (9 by default).

      -s,      to have only one solution.

      -v,      verbose output.

      -V,      display version and exit.

      -h,      display this help.

- exemple:    ./sudoku -g9

- enjoy.
